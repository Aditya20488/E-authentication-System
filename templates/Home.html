E-AUTHENTICATION SYSTEM USING OTP
AND QR CODE

----------------------------------------------------------------------------------------------

An E-Authentication System using OTP (One-Time Password) and QR (Quick Response) code is a two-way authentication system designed to enhance the security of user authentication.

The system requires the user to provide both a unique OTP generated by the system and a QR code generated and displayed on the user's device, which the system scans to confirm the user's identity
